# Minecraft Mod Translator Gemini

[English](README.md) | [한국어](README_KO.md)

이 도구는 마인크래프트 모드의 언어 파일을 번역해 주는 프로그램입니다.
모드 안의 글자를 읽어서 Gemini로 번역한 뒤, 번역된 새 파일을 만들어 줍니다.

## 할 수 있는 일

- 마인크래프트 모드 JAR 또는 폴더 열기
- 언어 파일 찾기
- Gemini로 번역하기
- 번역된 새 JAR 만들기

## 사용 방법

1. 내 컴퓨터에 맞는 버전을 다운로드합니다.
2. 프로그램을 엽니다.
3. 모드 파일 또는 폴더를 선택합니다.
4. 번역할 언어를 고릅니다.
5. 번역을 시작합니다.
6. 새 파일을 저장합니다.

## 참고

- 원본 파일은 바뀌지 않습니다.
- Gemini API 키가 필요합니다.
- 자세한 설명은 개발자 문서를 보세요.
