# Minecraft Mod Translator Gemini

[English](README.md) | [한국어](README_KO.md)

This tool helps translate Minecraft mod language files.
It reads the text inside a mod file, sends it to Gemini, and makes a translated copy.

## What it does

- Opens a Minecraft mod JAR or folder
- Finds language files
- Translates text with Gemini
- Makes a new translated JAR

## How to use

1. Download the release for your system.
2. Open the app.
3. Choose the mod file or folder.
4. Choose the target language.
5. Start translation.
6. Save the new file.

## Notes

- The original mod file is not changed.
- You need a Gemini API key.
- More details are in the developer guide.
